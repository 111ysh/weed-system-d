# 处方图格式标准文档

#### 1.投影坐标系：

```markdown
跟着WeedsGalore数据集来
```

####  2.元数据必填字段（处方图要包含的信息）

- ```markdown
  model_version：暂时写“待更新”
  ```

- ```markdown
  generate_time：生成时间（比如“2024-XX-XX XX:XX:XX”，后续脚本会自动填）
  ```

- ```markdown
  weed_density_threshold：5株/㎡（就用这个默认值，后续能改）
  ```

- ```markdown
  projection：EPSG:32610（农机常用）
  ```

#### 3.像素值编码（处方图每个点代表啥）

- 0：不用除草

- 1：轻度除草

- 2：重度除草

   

#### 4.分辨率

```markdown
1024×1024像素
```
